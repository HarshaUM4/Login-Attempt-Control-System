Login Attempt Control System
A simple Python project that controls login access by limiting the number of incorrect password attempts.
Features
Password verification
Limited login attempts
Temporary account lock after multiple failures
Easy to understand and implement
Requirements
Python 3.x
Run the Program
Bash
python login_attempt_control.py
Example
Plain text
Enter password: test
Incorrect password! Attempts remaining: 2

Enter password: admin123
Login Successful!
Purpose
This project demonstrates a basic security mechanism used to protect systems from unauthorized access.
